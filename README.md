# API Node without dependencies

### This project was created with the purpose of studies

In it I try to redeploy features of the express framework. 
Using ECMAScript Modules throughout the project.

### To run this project:

`$ npm start`

OR

`$ PORT=8080 npm start`

### Default routes are: 

**METHOD**: *GET*
- `/users`
- `/users/:id`

**METHOD**: *POST*
- `/users`
